Keycloak Java Adapter Configuration

2017-01-05
Java, Keycloak

@ keycloak adapterеҸҜд»ҘеңЁwarеҢ…йҮҢж·»еҠ keycloak.jsonй…ҚзҪ®пјҢд№ҹеҸҜд»ҘеңЁstandalone.xmlйҮҢй…ҚзҪ®гҖӮ ејҖеҸ‘иҝҮзЁӢжҺЁиҚҗдҪҝз”Ёkeycloak.jsonпјҢз”ҹдә§зҺҜеўғжҺЁиҚҗдҪҝз”Ёstandalone.xmlгҖӮ дёӢйқўжҳҜstandaloneзҡ„й…ҚзҪ®ж јејҸпјҡ
<subsystem xmlns="urn:jboss:domain:keycloak:1.1"></subsystem>

д»ҘдёӢжҳҜй…ҚзҪ®зҡ„еұһжҖ§

@ resource, еә”з”Ёзҡ„client idгҖӮRequired

@ realm-public-key, PEMж јејҸзҡ„realm public keyгҖӮеҸҜд»Ҙд»ҺconsoleиҺ·еҸ–пјҢдёҚе»әи®®и®ҫзҪ®гҖӮеҰӮжһңдёҚи®ҫзҪ®пјҢKeycloakдјҡжҜҸж¬ЎдёӢиҪҪж–°зҡ„public keyгҖӮ

@ auth-server-urlпјҢKeycloakжңҚеҠЎеҷЁзҡ„urlгҖӮжүҖжңүKeycloakйЎөйқўе’ҢRESTжңҚеҠЎйғҪжҳҜд»ҺиҝҷдёӘең°еқҖиҪ¬еҸ‘зҡ„гҖӮ http://host:port/authгҖӮ Required

@ ssl-requiredпјҢзЎ®дҝқжүҖжңүжІҹйҖҡйғҪйҖҡиҝҮHTTPSпјҢеңЁдә§е“Ғдёӯеә”иҜҘи®ҫзҪ®дёәallгҖӮй»ҳи®ӨеҖјexternalпјҢж„ҸжҖқHTTPSеҜ№еӨ–йғЁиҜ·жұӮжҳҜеҝ…йЎ»зҡ„гҖӮеҸҜз”ЁеҖјall, external, none.

@ use-resource-role-mappingsпјҢеҰӮжһңжҳҜtrueпјҢadapterдјҡеңЁtokenйҮҢжүҫapplicationзә§еҲ«зҡ„role mappingsгҖӮеҰӮжһңжҳҜfalse, дјҡзңӢrealmзә§еҲ«зҡ„role mappingsгҖӮй»ҳи®ӨжҳҜfalse

@ public-client, еҰӮжһңжҳҜtrueпјҢйҖӮй…ҚеҷЁдёҚдјҡдёәclientеҗ‘KeycloakеҸ‘йҖҒиҜҒд№ҰгҖӮй»ҳи®ӨжҳҜfalse

@ enable-corsпјҢй»ҳи®ӨеҖјfalseгҖӮеҰӮжһңжҳҜtrueпјҢдјҡеӨ„зҗҶCORS еүҚзҪ®иҜ·жұӮпјҢеҗҢж—¶дјҡд»Һaccess tokenжЈҖжҹҘvalid originsгҖӮ

@ cors-max-ageгҖӮеҰӮжһңCORSжҳҜenabledпјҢиҝҷдёӘеҖјдјҡиў«и®ҫзҪ®еҲ°Access-Control-Max-AgeеӨҙдёӯгҖӮеҰӮжһңжІЎи®ҫзҪ®пјҢиҝҷдёӘеӨҙи®ҫе®ҡе°ұдёҚдјҡеңЁCORSе“Қеә”йҮҢгҖӮ

@ cors-allowed-methods, еҰӮжһңcorsејҖеҗҜпјҢи®ҫе®ҡAccess-Control-Allow-Methods

@ bearer-only, й’ҲеҜ№servicesпјҢеә”иҜҘи®ҫдёәtrueгҖӮеҰӮжһңејҖеҗҜпјҢadapterдёҚдјҡжҺҲжқғз»ҷз”ЁжҲ·пјҢеҸӘдјҡжЈҖжҹҘbearer tokensгҖӮй»ҳи®ӨжҳҜfalse

@ autodetect-bearer-only, еҰӮжһңеә”з”ЁжңҚеҠЎеҷЁж—ўжҳҜwebеә”з”ЁпјҢеҸҲжҳҜwebжңҚеҠЎпјҲSOAP or RESTпјүпјҢе°ұеә”иҜҘи®ҫдёәtrueгҖӮиҝҷж ·дјҡеҜ№жңӘжҺҲжқғзҡ„з”ЁжҲ·и·іиҪ¬еҲ°зҷ»еҪ•йЎөпјҢеҜ№жңӘжҺҲжқғзҡ„SOAP/RESTе®ўжҲ·з«Ҝдјҡиҝ”еӣһ401гҖӮKeycloakж №жҚ®еӨҙX-Requested-With, SOAPActionжҲ–AcceptжқҘзЎ®е®ҡжҳҜSOAPиҝҳжҳҜRESTе®ўжҲ·з«ҜгҖӮй»ҳи®ӨеҖјfalse

@ enable-basic-auth, е‘ҠиҜүadapterж”ҜжҢҒbasicйӘҢиҜҒгҖӮеҰӮжһңејҖеҗҜпјҢеҝ…йЎ»жҸҗдҫӣsecretгҖӮй»ҳи®ӨеҖјfalseгҖӮ

@ expose-tokenпјҢеҰӮжһңжҳҜtrueпјҢиў«жҺҲжқғзҡ„жөҸи§ҲеҷЁе®ўжҲ·з«ҜпјҲйҖҡиҝҮJS HTTPи°ғз”ЁпјүеҸҜд»ҘйҖҡиҝҮroot/k_query_bearer_tokenиҺ·еҸ–signed access tokenгҖӮй»ҳи®ӨжҳҜfalse

@ credentials, зЎ®е®ҡеә”з”Ёзҡ„еҮӯиҜҒгҖӮиҝҷжҳҜдёӘеҜ№иұЎз¬ҰеҗҲпјҢkeyжҳҜcredentialзұ»еһӢпјҢvalueжҳҜcredentialзұ»еһӢзҡ„еҖјгҖӮзҺ°йҳ¶ж®өж”ҜжҢҒpasswordе’ҢjwtгҖӮеҜ№вҖҳConfidentialвҖҷи®ҝй—®ж–№ејҸжқҘиҜҙжҳҜRequired

@ connection-pool-size, AdapterдјҡеҲӣе»әйўқеӨ–зҡ„HTTPи°ғз”ЁKeycloak ServerжқҘи®Іaccess codeиҪ¬дёәaccess tokenгҖӮиҝҷдёӘй…ҚзҪ®еӨҡе°‘дёӘеҲ°Keycloak serverзҡ„иҝһжҺҘжұ иЈ…еӨҡе°‘дёӘиҝһжҺҘпјҢй»ҳи®ӨжҳҜ20

@ disable-trust-manager, еҰӮжһңKeycloak serverиҰҒжұӮHTTPSпјҢе№¶дё”иҝҷдёӘеұһжҖ§и®ҫзҪ®дёәtrueгҖӮйӮЈд№ҲдҪ е°ұдёҚйңҖиҰҒй…ҚзҪ®truststoreгҖӮиҝҷдёӘи®ҫзҪ®еҸӘз”ЁеңЁејҖеҸ‘жЁЎејҸдёӯпјҢproductionзҺҜеўғеә”иҜҘи®ҫзҪ®жҲҗfalseгҖӮй»ҳи®ӨжҳҜfalse

@ allow-any-hostname, еҰӮжһңKeycloak serverиҰҒжұӮHTTPSпјҢе№¶дё”иҝҷдёӘеұһжҖ§и®ҫзҪ®дёәtrueгҖӮйӮЈд№ҲKeycloak serverзҡ„йўҒеҸ‘иҜҒд№Ұе°ұйҖҡиҝҮtruststoreпјҢдҪҶжҳҜhost nameйӘҢиҜҒе°ұеҺ»жҺүдәҶгҖӮиҝҷдёӘи®ҫзҪ®еҸӘз”ЁеңЁејҖеҸ‘жЁЎејҸдёӯпјҢproductionзҺҜеўғеә”иҜҘи®ҫзҪ®жҲҗfalseгҖӮй»ҳи®ӨжҳҜfalse

@ truststore, жҢҮеҗ‘keystoreж–Үд»¶зҡ„и·Ҝеҫ„гҖӮеҰӮжһңеңЁи·Ҝеҫ„еүҚеҠ classpath:пјҢйӮЈд№Ҳtruststoreдјҡд»ҺйғЁзҪІзҡ„classpathжүҫж–Үд»¶гҖӮз”ЁдәҺеҗ‘еӨ–дёҺKeycloak serverеҒҡHTTPSжІҹйҖҡгҖӮе®ўжҲ·з«ҜеҲӣе»әHTTPSиҜ·жұӮйңҖиҰҒдёҖз§ҚйӘҢиҜҒжңҚеҠЎеҷЁдё»жңәзҡ„ж–№ејҸпјҢtruststoreе°ұжҳҜеҒҡиҝҷдёӘе·ҘдҪңзҡ„гҖӮkeystoreеҢ…еҗ«дёҖеҲ°еӨҡдёӘеҸ—дҝЎдё»жңәиҜҒд№ҰжҲ–еҮӯиҜҒз®ЎзҗҶдёӯеҝғгҖӮдҪ еҸҜд»Ҙи§ЈеҺӢKeycloak serverзҡ„SSL keystoreзҡ„public certificateжқҘеҲӣе»әtruststoreгҖӮйҷӨйқһssl-requiredжҳҜnoneжҲ–disable-trust-managerжҳҜtrueпјҢйӮЈд№ҲиҝҷдёӘеұһжҖ§е°ұжҳҜRequiredгҖӮ

@ truststore-password, truststore keystoreзҡ„еҜҶз ҒгҖӮеҰӮжһңи®ҫе®ҡдәҶtruststoreпјҢе№¶дё”truststoreйңҖиҰҒеҜҶз ҒпјҢйӮЈд№ҲиҝҷдёӘеұһжҖ§е°ұжҳҜRequiredгҖӮ

@ client-keystore, жҢҮеҗ‘keystoreзҡ„и·Ҝеҫ„гҖӮиҝҷдёӘkeystoreеҢ…еҗ«е®ўжҲ·з«ҜеҸҢеҗ‘SSLйӘҢиҜҒпјҢз”ЁдәҺadapterеҲӣе»әеҜ№Keycloak serverзҡ„HTTPSиҜ·жұӮгҖӮ

@ client-keystore-password, keystoreзҡ„еҜҶз ҒгҖӮеҰӮжһңclient-keystoreи®ҫе®ҡдәҶпјҢиҝҷдёӘе°ұжҳҜеҝ…йЎ»зҡ„гҖӮ

@ client-key-password, е®ўжҲ·з«Ҝkeyзҡ„еҜҶз ҒпјҢеҰӮжһңclient-keystoreи®ҫе®ҡдәҶпјҢиҝҷдёӘе°ұжҳҜеҝ…йЎ»зҡ„гҖӮ

@ always-refresh-tokenпјҢеҰӮжһңжҳҜtrueпјҢйҖӮй…ҚеҷЁдјҡеңЁжҜҸж¬ЎиҜ·жұӮж—¶йғҪеҲ·ж–°token пјҲиҝҷдёӘеңЁејҖеҸ‘жөӢиҜ•йҳ¶ж®өеҫҲжңүеҝ…иҰҒпјү

@ register-node-at-startup, еҰӮжһңжҳҜtureпјҢadapterдјҡжғіKeycloakеҸ‘йҖҒжіЁеҶҢиҜ·жұӮгҖӮиҝҷдёӘеұһжҖ§й»ҳи®ӨжҳҜfalseгҖӮеңЁclusterзҺҜеўғдёӯиө·дҪңз”ЁгҖӮ

@ register-node-period, е®ҡжңҹеңЁKeycloakйҮҚж–°жіЁеҶҢadapterгҖӮclusterзҺҜеўғдёӯдҪҝз”ЁгҖӮ

@ token-store, еҸҜд»ҘжҳҜsessionжҲ–cookieгҖӮй»ҳи®ӨжҳҜsessionпјҢadapterе°ҶиҙҰжҲ·дҝЎжҒҜеӯҳеӮЁеңЁHTTP sessionдёӯгҖӮ

@ principal-attribute, OpenID Connection ID Token attribute to populate the UserPrincipal name with. еҰӮжһңtokenеұһжҖ§жҳҜnullпјҢй»ҳи®ӨеҖје°ұжҳҜsub. еҸҜиғҪзҡ„еҖјжңүsub, preferred_username, email, name, nickname, given_name, family_name.

@ turn-off-change-session-id-on-login, й»ҳи®Өжғ…еҶөдёӢпјҢsession id дјҡеңЁзҷ»еҪ•жҲҗеҠҹеҗҺиў«ж”№еҸҳпјҢжқҘжҸ’е…ҘдёҖдёӘе®үе…Ёж”»еҮ»е®№еҷЁгҖӮи®ҫзҪ®жҲҗtrueпјҢеҰӮжһңдҪ жғіе…ій—ӯиҝҷдёӘеҠҹиғҪгҖӮй»ҳи®ӨеҖјжҳҜfalse

@ token-minimum-time-to-live, еӨ§еӨҡж•°жғ…еҶөдёӢпјҢдјҡеңЁaccess tokenеӨұж•ҲеүҚеҲ·ж–°гҖӮе°Өе…¶жҳҜе°Ҷaccess tokenеҸ‘йҖҒз»ҷREST clientпјҢtokenеҸҜиғҪдјҡеңЁйӘҢиҜҒе®ғд№ӢеүҚеӨұж•ҲпјҢжүҖд»ҘеҲ·ж–°еҫҲжңүеҝ…иҰҒгҖӮиҝҷдёӘеҖјдёҚеә”иҜҘеӨ§дәҺrealmвҖҷs access tokenеҜҝе‘ҪгҖӮй»ҳи®ӨжҳҜ0пјҲз§’пјүпјҢ жүҖд»ҘadapterдјҡеңЁе®ғеӨұж•Ҳж—¶еҲ·ж–°е®ғгҖӮ

@ min-time-between-jwks-request, еҗ‘KeycloakеҸ‘йҖҒиҜ·жұӮж–°зҡ„public keyзҡ„й—ҙйҡ”пјҢеҚ•дҪҚз§’гҖӮй»ҳи®ӨжҳҜ10гҖӮеҪ“adapterдёҚи®ӨиҜҶkidж—¶пјҢдјҡдёӢиҪҪе°қиҜ•ж–°зҡ„public keyгҖӮ10з§’еҶ…дёҚдјҡйҮҚеӨҚиҜ·жұӮпјҢйҳІжӯўжңүDoSж”»еҮ»пјҢеҸ‘йҖҒи®ёеӨҡй”ҷиҜҜзҡ„kidгҖӮ

@ public-key-cache-ttl, еҗ‘KeycloakеҸ‘йҖҒиҜ·жұӮж–°зҡ„public keyзҡ„жңҖеӨ§й—ҙйҡ”пјҢй»ҳи®ӨжҳҜ86400з§’пјҲ1еӨ©пјүгҖӮеҪ“adapterдёҚи®ӨиҜҶkidж—¶пјҢдјҡдёӢиҪҪе°қиҜ•ж–°зҡ„public keyгҖӮеҰӮжһңи®ӨиҜҶkidпјҢдјҡдҪҝз”Ёе·ІеӯҳеңЁзҡ„public keyгҖӮи¶…иҝҮиҝҷдёӘжңҹйҷҗпјҢдјҡдёӢиҪҪж–°зҡ„keyгҖӮ
