2). PSQLException: ERROR: column “user_id” contains null values"
由于hibernate4对表关联创建的列为users_id和roles_id，升级为hibernate5后，hibernate尝试将列名改为user_id，roles_id，而在进行alter table user_roles add column User_id int8 not null操作时，遇到非空检查失败了。解决方法是将原表备份，然后删除user_roles表，启动项目后会自动创建新的表和表结构。然后再自行更新表。
